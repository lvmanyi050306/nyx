Nyx Density Explorer：宇宙密度演化交互式可视分析系统

本作品基于 Nyx 宇宙学模拟密度数据，围绕三维密度场随时间演化展开可视分析。系统通过体数据投影、密度直方图、 时间-密度热力图、统计曲线和密度区间联动刷选，展示宇宙密度从相对均匀到高低密度分化、丝状结构和节点逐渐显现的过程。
