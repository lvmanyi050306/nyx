## 【第一题】数据读取与体绘制：如何展示宇宙演化中密度信息的变化？

Nyx 原始数据为 little-endian float32 格式的 `.dat` 二进制体数据，不能直接作为图片查看。本作品首先根据文件大小推断三维网格尺寸，将 z-y-x 存储顺序恢复为 x-y-z 体数据，再通过中心切片检查读取方向。随后使用 log-density 与百分位归一化压缩动态范围，并采用自定义 alpha compositing、传递函数和梯度增强进行体绘制，从空间形态上展示宇宙密度结构随时间演化的变化。

●过程：
（1）读取 `.dat` 文件中的 little-endian float32 数据，并根据体素数量推断 `n×n×n` 网格。
（2）按照 z 轴最快、y 轴其次、x 轴最慢的存储顺序恢复三维体数据。
（3）提取 XY、XZ、YZ 三个中心切片，验证读取方向和轴顺序是否合理。
（4）对 density 进行 `log10(max(density, eps))` 变换，并使用 P5-P99.7 百分位归一化。
（5）使用自定义 alpha compositing、LUT、smoothstep 透明度函数和梯度增强生成关键帧体绘制图。

●发现：
（1）中心切片呈现连续纹理，说明数据读取和三维恢复基本正确。
（2）t=0000 时密度结构相对连续，高密度节点不明显；t=0060 和 t=0099 中丝状连接与局部团块更加清晰。
（3）传递函数改变了可观察重点：低密度空洞、中密度丝状结构和高密度节点可以分别被突出显示。

## 【第二题】宇宙密度时序统计特征分析：密度分布随时间如何变化？

为避免只依赖单帧图像，本作品遍历全部时间步，计算 mean、std、max、P01、P05、P50、P95、P99、P99/mean 和 P99-P01 等指标，并绘制代表时间步 log-density histogram 与 time-density heatmap。统计结果用于从数值分布角度观察密度场由相对集中到逐渐分化的变化过程。

●过程：
（1）对 100 个时间步逐帧读取密度体数据，并展开为体素数组。
（2）计算全局统计量和百分位指标，保存为 `density_stats.csv`。
（3）对密度进行 log-density 变换，构建代表时间步的对数密度直方图。
（4）使用统一 bins 叠加全部时间步的 histogram，生成 time-density heatmap。
（5）通过统计曲线追踪 std、P99、P99/mean 和 P99-P01 的变化。

●发现：
（1）早期时间步密度分布更集中，后期分布逐渐变宽。
（2）高密度右尾增强，说明局部极高密度体素逐渐突出。
（3）mean 变化不一定最明显，std、P99/mean 和 P99-P01 更能体现密度不均匀性增强。
（4）统计趋势与体绘制中高密度节点逐渐突出的现象相互印证。

## 【第三题】Top 1% 高密度区域验证：高密度尾部是否具有空间结构意义？

本作品使用每个时间步原始 density 的 P99 作为 top 1% 高密度阈值，生成高密度 mask，并通过 X/Y/Z 三方向 MIP 和 P90/P95/P99 多阈值等值面观察空间分布。该分析直接回答直方图高密度尾部是否对应空间中的宇宙网致密节点。

●过程：
（1）选取每个时间步密度分布的 P99 作为极高密度阈值。
（2）筛选 `density >= P99` 的体素，构建 top 1% 高密度 mask。
（3）分别沿 X、Y、Z 三个方向进行最大强度投影，观察高密度体素的空间位置。
（4）使用 P90、P95、P99 多阈值等值面观察结构层级。
（5）将筛选结果与直方图右尾和体绘制结果进行对照。

●发现：
（1）P99 以上体素并非随机分散，而是集中在若干高密度节点和丝状结构交汇区域。
（2）P90、P95、P99 等值面呈现由外围结构到核心结构逐渐收缩的层级关系。
（3）高密度右尾在空间中对应宇宙网的致密节点区域。
（4）P99 筛选结果为体绘制中高亮结构提供了统计依据。

## 【第四题】相空间交互式刷选分析：如何实现统计特征与空间结构的双向关联？

本作品构建了 linked brushing 仪表盘和 HTML Web 展示系统。用户可以在 log-density histogram 或密度百分位滑条中选择密度区间，系统实时生成 mask 并映射回空间投影视图。当选择 99%-100% 区间时，页面会突出显示 top 1% 高密度体素，用于交互验证高密度尾部与空间节点结构之间的关系。

●过程：
（1）构建 log-density histogram 作为统计分布视图。
（2）设置 density percentile range，用于选择指定密度区间。
（3）根据用户选择实时生成 mask，并映射回三维体数据空间。
（4）在空间投影视图中显示被选体素，支持 MIP、Mask、Masked MIP 和 Slice。
（5）在 Web 页面中加入 time slider、Play/Pause、投影方向切换和 Top 1% 一键筛选。

●发现：
（1）普通密度区间可用于观察不同密度层级的空间分布范围。
（2）选择 99%-100% 时，空间中突出显示的区域集中于高密度节点和丝状交汇处。
（3）直方图刷选与空间联动使用户可以从统计分布反查空间结构。
（4）Web 展示系统比静态图更适合答辩演示和交互探索。

## 【第五题】综合结论：宇宙密度演化规律、可视化技术价值与实验挑战是什么？

综合体绘制、统计曲线、time-density heatmap、P99 高密度筛选和 linked brushing 结果，Nyx 密度场在课程实验可视化层面呈现由相对均匀到逐渐分化的趋势。高密度尾部增强，低密度区域与高密度节点逐渐分离，丝状结构和团块化特征更加明显。

●过程：
（1）从数据读取、中心切片、统计分析逐步扩展到体绘制和结构验证。
（2）利用 P99 筛选把密度分布右尾映射回空间节点。
（3）利用 Web 系统实现时间播放、密度刷选和空间联动展示。
（4）使用 density-gradient、Hessian 分类和时间步相似性矩阵作为进阶补充分析。

●发现：
（1）早期密度分布相对集中，后期密度波动增强。
（2）高密度尾部增强，P99、P99/mean 与 P99-P01 可作为团块化观察指标。
（3）P99 高密度体素集中在丝状结构交汇和节点区域。
（4）体绘制、统计曲线、time-density heatmap 和 P99 筛选结果相互印证。
（5）Web 交互系统进一步支持用户从统计分布主动探索空间结构。
