## 二、代码说明新增内容

### `code/step8_density_gradient_phase_space.m`

功能：构建 normalized log-density 与 gradient magnitude 的二维相空间热力图，并输出典型区域的空间 MIP。

输入：`data/raw/` 中的 Nyx `.dat` 文件。

输出：`results/08_density_gradient_phase_space/` 下的单帧相空间图、关键帧对比图和 `phase_space_selection_t0099.png`。

对应报告章节：3.10、4.8。

说明：该模块将一维密度直方图扩展为二维特征空间。横轴表示归一化后的 log-density，纵轴表示归一化后的梯度幅值。高梯度区域通常对应结构边界，低梯度区域则更可能对应平滑背景或致密核心内部。

### `code/step9_hessian_cosmic_web_classification.m`

功能：基于 Hessian 特征值进行 Void、Sheet、Filament、Node 粗分类，输出分类切片图、结构占比 CSV 和占比曲线。

输入：`data/raw/` 中的 Nyx `.dat` 文件。

输出：`results/09_hessian_cosmic_web/` 下的分类切片图、占比曲线和 Filament+Node MIP；同时输出 `data/processed/hessian_cosmic_web_fraction.csv`。

对应报告章节：3.11、4.9。

说明：该模块先对 log-density 进行平滑，并在必要时降采样，再计算三维 Hessian 特征值。通过统计小于 `-tau` 的特征值数量，将体素粗略分为 Void、Sheet、Filament 和 Node。该分类属于课程可视化实验中的近似结构识别，不作为严格天体物理结论。

### `code/step10_time_similarity_stage_analysis.m`

功能：基于全时间步 log-density histogram 计算时间步相似性矩阵，输出演化阶段划分图和 stage CSV。

输入：`data/raw/` 中的全部非空 Nyx `.dat` 文件。

输出：`results/10_time_similarity/` 下的相似性矩阵、距离矩阵、相邻变化曲线和阶段划分图；同时输出 `data/processed/time_similarity_stage.csv`。

对应报告章节：3.12、4.10。

说明：该模块将每个时间步的 log-density histogram 视为概率向量，使用 cosine similarity 衡量时间步之间的分布相似性，并根据 distance from start 对演化过程进行四阶段划分。

## 三、技术报告新增文字

### 3.10 density-gradient 二维相空间分析

**方法目的。** 原有 log-density 直方图能够展示密度数值分布，但它只包含单变量信息，无法区分“高密度平滑核心”和“中高密度剧烈变化边界”等不同空间结构。为增强对宇宙网边界和团块边缘的表达，本作业进一步引入 gradient magnitude，构建 density-gradient 二维相空间。

**方法思路。** 对每个代表时间步，首先计算 `log10(max(density, eps))`，再使用 P5 到 P99.7 百分位进行归一化，得到 normalized log-density。随后对归一化体数据计算三维梯度幅值，并对梯度幅值使用 P1 到 P99 百分位归一化。最终将每个体素表示为二维点 `(normalized log-density, normalized gradient magnitude)`，通过二维 histogram 形成相空间热力图。

**实现步骤。** 本模块选取 t=0000、0030、0060、0099 四个代表时间步；若目标时间步不存在，则自动选择最接近的已有文件。二维 histogram 使用 120×120 bins，颜色采用 `log10(voxel count + 1)`，以避免低频区域完全不可见。图中进一步标注四类典型区域：低密度低梯度的 void background、中密度高梯度的 filament boundary、高密度高梯度的 node boundary，以及高密度低梯度的 dense core。

**图像解释方式。** 相空间中的高梯度区域通常对应密度变化剧烈的位置，例如空洞边界、丝状结构边缘或团块边缘；高密度低梯度区域则可能对应致密核心内部。与一维密度直方图相比，该方法能够同时表达密度强度和局部变化程度，因此更适合支持二维传递函数设计和进阶 linked brushing。

**与 linked brushing 的关系。** 原有仪表盘主要基于密度百分位区间进行一维刷选，而 density-gradient 相空间可以将刷选条件扩展为二维特征区间。例如，用户不仅可以选择“高密度区域”，还可以进一步区分“高密度边界”和“高密度核心”。这为后续构建更丰富的交互式相空间刷选提供了基础。

### 3.11 基于 Hessian 特征值的宇宙网结构分类

**Hessian 的含义。** Hessian 矩阵描述标量场在局部空间中的二阶变化。对于 Nyx log-density 场，Hessian 特征值可以反映局部密度在不同方向上的曲率趋势。若多个方向上出现负曲率，说明该位置更接近局部汇聚或峰状结构。

**特征值与局部结构关系。** 本作业采用简化分类规则：设阈值 `tau`，统计三个 Hessian 特征值中小于 `-tau` 的数量，记为 `collapseDim`。当 `collapseDim=0` 时分类为 Void；当 `collapseDim=1` 时分类为 Sheet；当 `collapseDim=2` 时分类为 Filament；当 `collapseDim=3` 时分类为 Node。该规则可解释为：负曲率方向越多，局部结构越趋向多方向汇聚。

**实现步骤。** 为降低噪声影响，模块先对 log-density 做三维高斯平滑；如果 `imgaussfilt3` 不可用，则自动降级为 `smooth3` 或 separable convolution。为控制计算量，体数据会降采样到不超过 64×64×64。随后通过两次 `gradient` 计算 Hessian 的六个独立分量，并用向量化解析公式计算 3×3 对称 Hessian 的三个特征值。

**方法局限。** 本模块的分类结果依赖平滑尺度、降采样尺度和阈值 `tau`。由于这里只使用 density 单变量进行简化结构识别，结果主要服务于课程可视化实验中的空间结构解释，不应被表述为严格宇宙学科学分类结论。

### 3.12 基于密度分布相似性的时间阶段划分

**方法目的。** time-density heatmap 能展示密度分布随时间迁移和扩散，但它不能直接回答“哪些时间步彼此相似”以及“演化是否存在阶段性”。因此，本作业将每个时间步的 log-density histogram 向量化，并计算时间步之间的相似性矩阵。

**方法思路。** 对所有时间步使用统一 bins 计算 log-density histogram，并将每个 histogram 归一化为 probability 向量 `H(t,:)`。随后计算任意两个时间步之间的 cosine similarity，得到相似性矩阵 `S(i,j)`，其中数值越接近 1 表示两个时间步的整体密度分布越相似。距离矩阵定义为 `D=1-S`。

**阶段划分。** 模块进一步计算每个时间步与 t0000 的距离 `distance_from_start`，并按其 25%、50%、75% 分位将时间过程划分为 Early、Transition、Structure enhancement 和 Late 四个阶段。如果分位划分不稳定，则自动退化为按时间均分四段。

**方法意义。** 相似性矩阵属于时序数据可视分析方法，它不直接解释物理机制，但能够为代表时间步选择、阶段性叙事和可视化结果组织提供依据。当矩阵呈现沿对角线的块状结构时，说明相邻时间段内部更相似，而不同阶段之间的分布差异更明显。

### 4.8 density-gradient 二维相空间结果分析

图 `density_gradient_phase_compare.png` 展示了四个代表时间步的 density-gradient 二维相空间。横轴为 normalized log-density，纵轴为 normalized gradient magnitude，颜色表示 `log10(voxel count + 1)`。该图说明，大量体素集中在低到中等密度、低到中等梯度区域，而高密度或高梯度区域对应更少但更具结构意义的体素。

图 `phase_space_selection_t0099.png` 进一步展示了 t0099 中四类相空间区域的空间 MIP。void background 主要对应低密度、低梯度区域；filament boundary 体现中密度但局部变化较强的结构边界；node boundary 对应高密度且梯度较强的位置；dense core 则对应高密度但内部变化较平缓的致密区域。该结果说明，二维相空间能把“数值分布”和“空间结构类型”更细致地联系起来。

### 4.9 Hessian 宇宙网分类结果分析

图 `hessian_class_slice_compare.png` 展示了四个代表时间步中心切片的 Hessian 分类结果。Void、Sheet、Filament 和 Node 使用离散颜色表示。与体绘制不同，Hessian 分类并不直接追求真实渲染效果，而是尝试根据局部二阶结构将密度场划分为不同结构类别。

图 `hessian_class_fraction_curve.png` 给出了四类结构体素占比随时间的变化。该曲线可与体绘制、P99 筛选和结构指标曲线共同分析。如果 Filament 和 Node 的占比或空间连续性增强，则可以作为宇宙网结构逐渐显著的补充证据。图 `filament_node_mip_t0099.png` 将 Filament 与 Node 合并后做 MIP，用于观察中高密度连续结构和节点区域的空间分布。

需要强调的是，该 Hessian 分类是课程作业中的简化结构识别方法，结果受平滑尺度、阈值和降采样影响，不能作为正式天体物理分类结论。

### 4.10 时间步相似性与演化阶段分析

图 `time_step_similarity_matrix.png` 展示了 100 个时间步之间的 log-density histogram 余弦相似性。矩阵对角线附近相似度最高，说明相邻时间步的密度分布变化较连续；远离对角线的区域相似度降低，说明早期和后期分布存在差异。

图 `adjacent_time_change_curve.png` 展示相邻时间步之间的分布变化强度。该曲线可以用于识别变化较快的时间段。图 `evolution_stage_segmentation.png` 根据 distance from start 对时间序列进行四阶段划分，为报告中使用 t=0000、0030、0060、0099 作为代表时间步提供了数据依据：这些时间步分别覆盖了早期、过渡期、结构增强期和后期阶段。

### 第 5 章结论与展望补充

进阶方法带来的改进主要体现在三个方面。第一，density-gradient 相空间使交互刷选从一维密度区间扩展到二维特征区间，可以区分结构边界和致密核心。第二，Hessian 分类使宇宙网结构从视觉观察进一步扩展为局部结构识别，为 Void、Sheet、Filament 和 Node 的比较提供了近似依据。第三，时间步相似性矩阵为演化阶段划分和代表帧选择提供了数据支持，使报告叙事不只依赖主观选帧。

同时，本作业仍存在局限。Hessian 分类依赖平滑尺度和阈值 `tau`；相似性矩阵基于统计分布，仍然不包含完整空间对应关系；二维相空间刷选目前主要输出 MIP，不能完全替代真实三维交互探索。因此，本文结论应理解为数据可视化课程实验层面的分析结果，而不是正式天体物理科学结论。
