# Nyx 可视化项目代码说明

## 1. 代码总体结构

本项目采用“主入口 + 分步骤脚本 + 工具函数”的结构组织 MATLAB 代码。主入口文件为 code/main.m，负责自动设置项目根目录、添加路径、检查数据文件、创建输出目录，并按顺序调用各个分析步骤。每个 step 文件对应报告中的一个主要分析模块，utils 目录保存通用函数。

## 2. 主入口

### code/main.m

功能：

1. 自动识别 projectRoot；
2. 添加 code/ 和 code/utils/ 到 MATLAB 路径；
3. 检查 data/raw/ 是否存在非空 .dat 文件；
4. 依次运行 step1 至 step6；
5. 生成 results/frames/ 下的视频帧；
6. 提示用户手动运行 step7_interactive_dashboard。

运行方式：

```matlab
run('code/main.m')
```

## 3. 数据读取相关代码

### code/read_nyx_dat.m

功能：读取单个 Nyx .dat 文件并恢复三维体数据。

关键点：

- 使用 fopen(filename,'r','ieee-le') 读取 little-endian 数据；
- 使用 fread(...,'single=>single') 读取 float32；
- 调用 infer_grid_size 自动推断 n；
- 先 reshape 为 [z,y,x]，再 permute 为 [x,y,z]；
- 输出 single 类型三维数组。

### code/infer_grid_size.m

功能：根据体素总数推断 n，使 numel(data)=n^3。如果不能组成立方体网格，则报错提示文件尺寸异常。

## 4. 分步骤分析代码

### code/step1_check_data.m

功能：数据完整性检查。

输出：

- results/01_data_check/center_slices_0000.png
- results/01_data_check/file_size_check.png
- results/01_data_check/data_check_stats.mat

主要检查内容：

- 数据尺寸；
- min、max、mean、std；
- NaN、Inf、负值数量；
- XY、XZ、YZ 三方向中心切片；
- 所有 .dat 文件大小是否一致。

### code/step2_statistics_all_frames.m

功能：全时间步密度统计分析。

输出：

- data/processed/density_stats.csv
- data/processed/statistics.mat
- data/processed/high_density_threshold.csv
- results/03_statistics/mean_std_curve.png
- results/03_statistics/max_density_curve.png
- results/03_statistics/percentile_curve.png
- results/03_statistics/log_skew_kurtosis_curve.png

统计指标：

- mean_density、std_density、min_density、max_density；
- P01、P05、P50、P95、P99、P99.7、P99.9；
- mean_log_density、std_log_density；
- skew_log_density、kurtosis_log_density。

### code/step3_histogram_analysis.m

功能：对数密度直方图和时间-密度热力图分析。

输出：

- results/02_histograms/histogram_t0000.png
- results/02_histograms/histogram_t0030.png
- results/02_histograms/histogram_t0060.png
- results/02_histograms/histogram_t0099.png
- results/02_histograms/histogram_compare_keyframes.png
- results/02_histograms/time_density_heatmap.png

方法：

- 代表时间步使用 120 个 bins；
- histogram normalization 设置为 probability；
- 全时间步使用统一 bins 堆叠为二维热力图。

### code/step4_volume_render_keyframes.m

功能：代表时间步体绘制和传递函数对比。

输出：

- results/04_volume_render/volume_t0000.png
- results/04_volume_render/volume_t0030.png
- results/04_volume_render/volume_t0060.png
- results/04_volume_render/volume_t0099.png
- results/04_volume_render/volume_keyframes_compare.png
- results/04_volume_render/transfer_function_compare_t0099.png

方法：

- 对 density 进行 log10 变换；
- 使用 P5 到 P99.7 百分位归一化；
- 调用 volume_render_alpha_composite 做自定义体绘制；
- 比较空洞观察型、丝状结构型和高密度节点型三种传递函数。

### code/step5_high_density_selection.m

功能：P99 高密度区域筛选、最大强度投影和等值面绘制。

输出：

- results/05_high_density/top1_percent_t0000.png
- results/05_high_density/top1_percent_t0060.png
- results/05_high_density/top1_percent_t0099.png
- results/05_high_density/top1_percent_isosurface_t0099.png
- results/05_high_density/nested_isosurfaces_t0099.png

方法：

- 使用原始 density 的 P99 作为 top 1% 阈值；
- 生成三维 mask；
- 对 mask 区域 log-density 做 X/Y/Z 三方向 MIP；
- 使用 P90、P95、P99 绘制嵌套等值面。

### code/step6_structure_metrics.m

功能：结构指标计算和首末时间步差分分析。

输出：

- data/processed/structure_metrics.csv
- data/processed/structure_metrics.mat
- results/06_structure_metrics/spread_tail_curve.png
- results/06_structure_metrics/void_dense_fraction_curve.png
- results/06_structure_metrics/entropy_gini_curve.png
- results/06_structure_metrics/high_density_components_curve.png
- results/06_structure_metrics/difference_mip_first_last.png

指标：

- spread_p99_p01；
- tail_amplification；
- void_fraction_vs_t0000_p05；
- dense_fraction_vs_t0000_p95；
- log_density_entropy；
- gini_density；
- high_density_component_count；
- high_density_largest_component_size；
- high_density_mean_component_size。

### code/step7_interactive_dashboard.m

功能：交互式 linked brushing 仪表盘。

输出：

- results/07_dashboard/dashboard_preview.png

界面控件：

- 时间步滑条；
- 密度百分位下限滑条；
- 密度百分位上限滑条；
- 投影方向选择：X / Y / Z；
- 显示模式选择：MIP / Mask / LogDensity。

分析意义：左侧直方图用于选择密度区间，右侧空间投影视图显示被选体素的空间分布，从而建立统计分布与三维空间结构之间的联动关系。

## 5. 工具函数

### code/utils/ensure_folder.m

若目标目录不存在，则自动创建。

### code/utils/save_figure.m

统一保存 MATLAB figure，优先使用 exportgraphics，旧版本 MATLAB 自动降级为 print。

### code/utils/normalize_percentile.m

对输入数据进行百分位裁剪，并归一化到 [0,1]。主要用于体绘制和 MIP 显示。

### code/utils/smoothstep.m

平滑阶跃函数，用于透明度传递函数设计。

### code/utils/custom_nyx_colormap.m

自定义 Nyx 色图，从深蓝/黑色过渡到紫色、橙色、黄色和白色。

### code/utils/volume_render_alpha_composite.m

自定义体绘制核心函数。沿 z 方向从后向前逐层 alpha compositing，并结合梯度增强实现类光照效果。

### code/utils/gini_coefficient.m

计算原始密度的 Gini 系数，用于衡量密度不均匀程度。

## 6. 推荐答辩说明顺序

1. 先说明数据不是图片，而是 little-endian float32 三维体数据；
2. 说明如何通过文件大小推断 128×128×128 网格；
3. 展示中心切片，证明读取方向正确；
4. 展示体绘制关键帧，说明宇宙网结构逐渐形成；
5. 展示时间-密度热力图和统计曲线，说明密度分布逐渐扩散、高密度尾部增强；
6. 展示 P99 高密度筛选和多阈值等值面，说明右尾对应空间节点；
7. 展示 linked brushing 仪表盘，说明统计分布与空间结构可交互联动。
