## 摘要

宇宙大尺度结构的形成与演化是现代天体物理和科学可视化中的重要研究对象。Nyx 宇宙学模拟数据以三维密度场的形式记录物质分布随时间变化的过程，具有典型的体数据规模大、动态范围宽、时序帧数多和空间结构复杂等特点。本作业围绕 Nyx 二进制密度数据，设计并实现了一套从数据读取、统计分析、体绘制到交互刷选的完整可视分析流程。首先，本文根据 little-endian float32 格式读取 0000.dat 至 0099.dat 等时间步文件，通过自动推断网格尺寸和 z-y-x 到 x-y-z 的轴顺序重排恢复三维密度场；其次，对原始密度进行统计分析，并使用 log10 密度变换和百分位归一化压缩动态范围；随后，设计基于百分位传递函数的自定义体绘制方法，通过 alpha compositing、自定义颜色查找表和梯度增强突出宇宙网中的丝状结构、团块边界和高密度节点；此外，本文利用对数密度直方图、时间-密度热力图、P99 高密度筛选、多阈值等值面和 linked brushing 相空间刷选仪表盘，将统计分布与三维空间结构联系起来。实验结果表明，Nyx 密度场在时间演化过程中由较集中的密度分布逐渐发展出更强的密度波动，高密度尾部不断增强，P99 以上体素逐渐集中于丝状结构交汇处和高密度节点，低密度区域则表现为空洞扩张趋势，整体呈现出类似宇宙网结构形成的可视特征。本文工作主要面向数据可视化课程实践，用于展示体数据可视化、统计可视化和交互式联动分析方法在科学数据理解中的综合应用价值。

# 1 问题描述

## 1.1 研究背景

宇宙大尺度结构描述了星系、暗物质和气体等物质在宇宙空间中的整体分布形态。现代宇宙学认为，早期宇宙中的微小密度扰动会在引力作用下不断放大，逐渐形成空洞、丝状结构、片状结构以及高密度团块和节点。这些结构共同构成了通常所说的“宇宙网”。对宇宙网形成过程的观察和解释，不仅依赖天文观测，也依赖大规模数值模拟与科学可视化方法。

Nyx 是一种宇宙学流体模拟程序，其输出数据可以用三维密度场表示空间中物质分布的状态。本作业使用的 Nyx 数据由多个二进制 .dat 文件组成，每个文件对应一个时间步。每个时间步内部存储一个 n×n×n 的三维体数据，每个体素记录对应空间位置的密度值。与普通二维图像或表格数据不同，该数据同时具有三维空间维度和时间维度，因此适合采用体绘制、统计曲线、直方图热力图、最大强度投影和交互式刷选等方法进行综合分析。

## 1.2 数据可视化任务

本作业的目标不是对 Nyx 模拟进行严格天体物理参数推断，而是从数据可视化课程实践角度，构建一个完整、可复现、结构清晰的可视分析项目。具体而言，本作业希望回答：如何从二进制文件中正确恢复三维密度场；如何设计适合高动态范围密度数据的显示变换和传递函数；如何观察密度分布随时间从相对均匀到团块化、网络化的变化；如何验证直方图右侧高密度尾部是否对应空间中的宇宙网节点；以及交互式可视化技术如何帮助建立统计特征与空间结构之间的关联。

## 1.3 本作业拟解决的问题

本作业围绕以下五个问题展开：

1. 如何读取和恢复 Nyx 三维体数据？  
   原始 .dat 文件不是图像或表格，而是连续的 float32 数值。必须根据文件大小推断网格尺寸，并按照正确的轴顺序重排为 MATLAB 中直观的 x-y-z 三维数组。

2. 如何设计合适的传递函数展示密度结构？  
   原始密度动态范围较大，若直接线性显示，高密度区域会主导图像，中低密度丝状结构难以观察。因此需要结合 log10 变换、百分位裁剪、颜色查找表和透明度函数进行体绘制。

3. 密度分布随时间如何变化？  
   通过均值、标准差、最大值、百分位、偏度、峰度以及时间-密度热力图，可以观察密度分布是否扩散、高密度尾部是否增强，以及密度场是否逐渐两极分化。

4. 高密度尾部是否对应空间中的宇宙网节点？  
   本作业使用每个时间步原始 density 的 P99 作为 top 1% 高密度阈值，并在三维空间中生成 mask、MIP 和等值面，以验证统计分布右尾与空间节点和丝状交汇处之间的对应关系。

5. 可视化技术在宇宙学数据分析中有什么价值？  
   体绘制适合观察整体空间结构，统计图适合描述数值分布，交互式相空间刷选则可以在统计分布和空间位置之间建立联动关系。三者结合能够提升复杂科学数据的可解释性。

## 2.2 数据格式分析

Nyx 原始数据文件命名为 0000.dat、0001.dat、0002.dat，直至 0099.dat。每个文件表示一个时间步，内部不是图片，也不是带字段名的表格，而是一段连续的 float32 数值序列。每个 float32 占 4 字节，因此可以通过文件大小反推体素数量。例如，若文件大小为 8192 KB，即 8,388,608 字节，则体素数量为：

8,388,608 / 4 = 2,097,152 = 128 × 128 × 128

因此，该文件对应一个 128×128×128 的三维密度场。读取时需要注意原始线性顺序为 z 轴最快、y 轴其次、x 轴最慢。也就是说，连续数据首先沿 z 方向变化，然后沿 y 方向变化，最后沿 x 方向变化。在 MATLAB 中直接 reshape 后得到的是 [z,y,x] 顺序，因此需要再通过 permute 转换为更直观的 [x,y,z] 数组。

## 2.3 原始数据读取

原始数据读取过程的核心是二进制格式解析。程序使用 fopen(filename,'r','ieee-le') 指定 little-endian 字节序，并使用 fread 以 single 类型读取所有数据。读取完成后，根据 numel(data) 自动推断 n，要求 n^3 必须等于数据总量，否则说明文件尺寸异常。

核心读取步骤如下：

```matlab
fid = fopen(filename, 'r', 'ieee-le');
data = fread(fid, inf, 'single=>single');
n = infer_grid_size(numel(data));
Vz_y_x = reshape(data, [n, n, n]);
V = permute(Vz_y_x, [3, 2, 1]);
V = single(V);
```

该过程保证了三点：第一，字节序与数据格式匹配；第二，网格尺寸由文件自身自动推断，避免硬编码；第三，轴顺序从原始 z-y-x 转换为 MATLAB 中便于理解和可视化的 x-y-z。

## 2.4 数据完整性检查

在进行正式分析前，需要检查数据是否能够被正确读取。本作业首先读取 data/raw/ 下的 0000.dat，如果该文件不存在，则自动选择第一个非空 .dat 文件。检查指标包括三维尺寸、最小值、最大值、均值、标准差、NaN 数量、Inf 数量和负值数量。同时，程序遍历所有 .dat 文件并绘制文件大小曲线，用于判断各时间步文件大小是否一致。

为了验证空间轴恢复是否基本正确，本作业提取三维体数据的 XY、XZ、YZ 三个中心切片，并对 log10(max(V,eps)) 后的密度使用 imagesc 显示。若三个方向的中心切片均呈现连续纹理，且没有明显周期性条纹或错位现象，则说明二进制读取、网格尺寸推断和轴顺序恢复基本正确。若出现块状错位、拉伸条纹或完全无结构图像，则可能是数据类型、字节序或 reshape/permute 顺序错误。

## 2.5 对数变换与百分位归一化

Nyx 密度数据通常具有明显的高密度尾部，即少量体素可能具有远高于平均值的密度。如果直接以线性尺度显示，图像会被极高密度区域主导，而中低密度丝状结构和空洞边界难以辨认。因此，本作业在显示阶段采用 log10 变换：

```matlab
logV = log10(max(V, eps('single')));
```

其中 max(V,eps) 用于避免对零或极小值取对数导致数值问题。之后对 log-density 使用百分位裁剪与归一化。默认体绘制显示范围为 P5 到 P99.7，即去除极端低值和极端高值对显示范围的过度影响：

```matlab
Vn = normalize_percentile(logV, 5, 99.7);
```

需要强调的是，本作业在统计计算和高密度阈值筛选时仍使用原始 density，而在图像显示、体绘制和投影可视化时使用 log-density。这种区分可以兼顾数值统计的物理意义和可视表达的清晰度。

## 3.2 密度时序统计分析方法

方法目的：从数值统计角度刻画 Nyx 密度场随时间的整体演化趋势。

方法思路：对 data/raw/ 下所有非空 .dat 文件按文件名排序，并逐帧读取三维体数据。每个时间步将三维体数据展平成一维体素数组，计算原始 density 的均值、标准差、最小值、最大值和多个百分位，同时计算 log-density 的均值、标准差、偏度和峰度。

计算指标包括：

- mean_density
- std_density
- min_density
- max_density
- P01、P05、P50、P95、P99、P99.7、P99.9
- mean_log_density
- std_log_density
- skew_log_density
- kurtosis_log_density

这些指标中，均值描述整体密度水平，标准差反映密度波动强度，高百分位和最大值反映高密度尾部，偏度和峰度反映 log-density 分布的不对称性和尖峰程度。实验结果输出为 density_stats.csv 和 statistics.mat，并进一步绘制统计曲线。

## 3.3 对数密度直方图与时间-密度热力图

方法目的：观察单个时间步的密度分布形态，以及全部时间步中密度分布随时间的迁移和扩散。

## 3.4 自定义体绘制与传递函数设计

方法目的：在二维图像中表达三维密度场的空间结构，观察空洞、丝状结构和高密度节点。

方法思路：本作业不直接依赖 MATLAB 的 volshow，而是实现自定义 alpha compositing。程序沿 z 方向从后向前逐层合成，将每一层的归一化密度值映射为颜色和透明度。颜色映射采用从深蓝/黑色到紫色、橙色、黄色和白色的自定义 LUT；透明度传递函数采用 smoothstep 控制平滑过渡。

传递函数设计遵循以下原则：低密度区域接近透明，避免遮挡整体结构；中密度区域半透明，用于展示宇宙网丝状连接；高密度区域透明度更高，用于突出节点和团块。本作业还额外设计了三种传递函数：空洞观察型、丝状结构型和高密度节点型，用于展示同一体数据在不同科学关注点下的可视差异。

## 3.5 梯度增强与类光照处理

方法目的：提高体绘制图像中结构边界、丝状连接和团块边缘的可辨识度。

方法思路：对归一化后的 log-density 体数据 Vn 计算三维梯度：

```matlab
[gx, gy, gz] = gradient(Vn);
gradMag = sqrt(gx.^2 + gy.^2 + gz.^2);
```

梯度幅值较大的区域通常对应密度变化较剧烈的位置，例如空洞边界、丝状结构外缘和团块边缘。程序对 gradMag 进行百分位归一化后，将其用于增强 alpha 或 brightness。该处理并不追求物理真实光照，而是一种视觉增强策略，目的是让报告图像中的空间结构更加清晰。

可视化解释时应注意，梯度增强突出的是密度变化边界，不代表真实光源照明结果。因此在科学报告中应将其描述为“类光照”或“结构增强”，而不是严格物理渲染。

## 3.6 高密度区域筛选与最大强度投影

方法目的：验证统计分布右侧高密度尾部是否在空间中对应宇宙网节点、团块和丝状交汇处。

方法思路：对每个代表时间步使用原始 density 的 P99 作为 top 1% 高密度阈值，生成三维 mask：

```matlab
thr = prctile(density, 99);
mask = V >= thr;
```

## 3.7 多阈值等值面与结构层级分析

方法目的：展示密度结构从大尺度丝状外壳向高密度核心节点收缩的层级关系。

方法思路：在 t=0099 或最后一个时间步中，分别计算原始 density 的 P90、P95 和 P99 阈值，并绘制三个等值面。其中 P90 表示中高密度丝状区域，P95 表示更集中的高密度聚集区域，P99 表示极高密度节点。三个阈值使用不同颜色和透明度显示。

结果解释时，P90 等值面通常覆盖范围较大，能够形成较连续的丝状外壳；P95 等值面范围缩小，集中于更强的聚集区域；P99 等值面只保留最致密节点。该图能够从空间层级角度解释“宇宙网”不是单一阈值结构，而是由不同密度层次共同构成。

## 3.8 结构指标计算与宇宙网演化分析

方法目的：在图像观察之外，用可量化指标描述密度场从相对均匀到团块化、两极分化的演化过程。

本作业计算的结构指标包括：

- spread_p99_p01 = P99 - P01，用于描述密度分布跨度；
- tail_amplification = P99 / mean_density，用于描述高密度尾部相对平均水平的增强；
- void_fraction_vs_t0000_p05，表示低于 t0000 P05 阈值的体素比例；
- dense_fraction_vs_t0000_p95，表示高于 t0000 P95 阈值的体素比例；
- log_density_entropy，用于描述 log-density 分布复杂度；
- gini_density，用于描述密度不均匀程度；
- high_density_component_count、largest_component_size 和 mean_component_size，用于描述 P99 高密度区域的连通结构。

这些指标可以从不同角度量化宇宙密度演化：spread 和 std 的增大表示密度波动增强；tail_amplification 的增大表示极端高密度区域变得更加突出；void_fraction 和 dense_fraction 同时变化说明低密度空洞和高密度团块并行发展；Gini 系数增大则反映密度分布更加不均匀。

方法思路：仪表盘左侧显示当前时间步的 log-density histogram，用户通过百分位下限和上限滑条选择密度区间。系统根据原始 density 阈值实时生成 mask，并在右侧显示对应的空间投影。用户还可以切换时间步、投影方向 X/Y/Z 和显示模式 MIP/Mask/LogDensity。界面同时显示被选体素数量和比例。

# 5 结论与展望

## 5.1 主要结论

通过对 Nyx 宇宙密度数据的读取、统计、体绘制和交互分析，本作业得到以下结论：

1. 早期宇宙密度分布相对集中，log-density 直方图较窄，体绘制中高密度节点和网络结构尚不明显。
2. 随时间推进，密度波动逐渐增强，标准差、高百分位和最大密度等指标更能反映结构演化。
3. 高密度尾部不断拉长，P99 以上体素逐渐集中在丝状结构交汇处和高密度节点。
4. 低密度区域逐渐扩张，形成空洞；高密度区域逐渐聚集，形成团块、节点和丝状连接。
5. 多阈值等值面显示出从 P90 丝状外壳到 P99 致密节点的层级结构，体现宇宙网结构的多尺度特征。
6. 体绘制适合观察三维空间结构，直方图和统计曲线适合观察数值分布演化，交互刷选适合建立统计特征与空间结构之间的关联。



## 5.2 方法优势与局限性

本作业的方法优势主要体现在以下方面。第一，项目形成了从数据读取、完整性检查、统计计算、体绘制、高密度筛选到交互分析的完整流程，便于复现实验结果。第二，自定义体绘制使颜色映射和透明度传递函数可控，能够针对空洞、丝状结构和高密度节点设计不同视觉强调方式。第三，log-density 与百分位归一化有效缓解了高动态范围密度数据的显示问题。第四，P99 筛选将统计分布右尾与空间节点联系起来，使直方图分析不再停留于纯数值层面。第五，linked brushing 仪表盘增强了探索性分析能力，使用户可以主动选择密度区间并观察其空间位置。

同时，本作业也存在一定局限。当前体绘制主要沿固定 z 方向合成，缺少真正自由的三维旋转和交互裁剪；梯度增强属于类光照处理，并不是真实物理光照模型；当前分析只使用 density 变量，没有结合 velocity、temperature 等多变量信息；P99 阈值具有经验性，不同阈值会影响结构范围；MIP 会压缩深度信息，不能完全替代真实三维交互。最后，本文结论主要用于数据可视化课程实验分析，不能直接作为正式天体物理科学结论。


## 5.3 方法价值

本作业将体数据可视化、统计可视化和交互式可视分析结合起来，形成了较完整的 Nyx 密度场分析流程。自定义体绘制避免了对单一黑箱可视化工具的依赖，使传递函数、颜色映射和透明度设计更加可控。P99 高密度筛选和多阈值等值面提供了从统计阈值到空间结构的解释路径。交互式 linked brushing 仪表盘则进一步增强了探索性分析能力。

## 5.4 不足与展望

本作业仍存在一些不足。首先，体绘制采用的是简化 alpha compositing 和梯度增强，并非严格物理光照模型；其次，分析主要围绕密度场展开，未结合速度场、温度场或其他 Nyx 物理变量；第三，结构指标偏向课程可视化实践，不能直接等同于正式天体物理结论。

未来可以从三个方向扩展：一是引入更高质量的光线投射与光照模型，提高体绘制真实感；二是结合多变量数据进行联合可视化，例如密度-温度、密度-速度关系；三是进一步增强交互系统，支持三维旋转、区域框选、时间动画播放和多视图联动，从而形成更完整的科学可视分析平台。

# 进阶可视分析新增章节

## 3.10 density-gradient 二维相空间分析

**方法目的。** 原有 log-density 直方图能够展示密度数值分布，但它只包含单变量信息，无法区分“高密度平滑核心”和“中高密度剧烈变化边界”等不同空间结构。为增强对宇宙网边界和团块边缘的表达，本作业进一步引入 gradient magnitude，构建 density-gradient 二维相空间。

**方法思路。** 对每个代表时间步，首先计算 `log10(max(density, eps))`，再使用 P5 到 P99.7 百分位进行归一化，得到 normalized log-density。随后对归一化体数据计算三维梯度幅值，并对梯度幅值使用 P1 到 P99 百分位归一化。最终将每个体素表示为二维点 `(normalized log-density, normalized gradient magnitude)`，通过二维 histogram 形成相空间热力图。

**实现步骤。** 本模块选取 t=0000、0030、0060、0099 四个代表时间步；若目标时间步不存在，则自动选择最接近的已有文件。二维 histogram 使用 120×120 bins，颜色采用 `log10(voxel count + 1)`，以避免低频区域完全不可见。图中进一步标注四类典型区域：低密度低梯度的 void background、中密度高梯度的 filament boundary、高密度高梯度的 node boundary，以及高密度低梯度的 dense core。

**图像解释方式。** 相空间中的高梯度区域通常对应密度变化剧烈的位置，例如空洞边界、丝状结构边缘或团块边缘；高密度低梯度区域则可能对应致密核心内部。与一维密度直方图相比，该方法能够同时表达密度强度和局部变化程度，因此更适合支持二维传递函数设计和进阶 linked brushing。

## 3.11 基于 Hessian 特征值的宇宙网结构分类

**Hessian 的含义。** Hessian 矩阵描述标量场在局部空间中的二阶变化。对于 Nyx log-density 场，Hessian 特征值可以反映局部密度在不同方向上的曲率趋势。若多个方向上出现负曲率，说明该位置更接近局部汇聚或峰状结构。

**分类规则。** 本作业采用简化分类规则：设阈值 `tau`，统计三个 Hessian 特征值中小于 `-tau` 的数量，记为 `collapseDim`。当 `collapseDim=0` 时分类为 Void；当 `collapseDim=1` 时分类为 Sheet；当 `collapseDim=2` 时分类为 Filament；当 `collapseDim=3` 时分类为 Node。该规则可解释为：负曲率方向越多，局部结构越趋向多方向汇聚。

**实现与局限。** 为降低噪声影响，模块先对 log-density 做三维高斯平滑；如果 `imgaussfilt3` 不可用，则自动降级为 `smooth3` 或 separable convolution。为控制计算量，体数据会降采样到不超过 64×64×64。该分类结果依赖平滑尺度、降采样尺度和阈值 `tau`，主要服务于课程可视化实验中的空间结构解释，不应被表述为严格宇宙学科学分类结论。

## 3.12 基于密度分布相似性的时间阶段划分

**方法目的。** time-density heatmap 能展示密度分布随时间迁移和扩散，但它不能直接回答“哪些时间步彼此相似”以及“演化是否存在阶段性”。因此，本作业将每个时间步的 log-density histogram 向量化，并计算时间步之间的相似性矩阵。

**方法思路。** 对所有时间步使用统一 bins 计算 log-density histogram，并将每个 histogram 归一化为 probability 向量 `H(t,:)`。随后计算任意两个时间步之间的 cosine similarity，得到相似性矩阵 `S(i,j)`，其中数值越接近 1 表示两个时间步的整体密度分布越相似。距离矩阵定义为 `D=1-S`。

**阶段划分。** 模块进一步计算每个时间步与 t0000 的距离 `distance_from_start`，并按其 25%、50%、75% 分位将时间过程划分为 Early、Transition、Structure enhancement 和 Late 四个阶段。如果分位划分不稳定，则自动退化为按时间均分四段。

## 第 5 章结论与展望补充

进阶方法带来的改进主要体现在三个方面。第一，density-gradient 相空间使交互刷选从一维密度区间扩展到二维特征区间，可以区分结构边界和致密核心。第二，Hessian 分类使宇宙网结构从视觉观察进一步扩展为局部结构识别，为 Void、Sheet、Filament 和 Node 的比较提供了近似依据。第三，时间步相似性矩阵为演化阶段划分和代表帧选择提供了数据支持，使报告叙事不只依赖主观选帧。

同时，本作业仍存在局限。Hessian 分类依赖平滑尺度和阈值 `tau`；相似性矩阵基于统计分布，仍然不包含完整空间对应关系；二维相空间刷选目前主要输出 MIP，不能完全替代真实三维交互探索。因此，本文结论应理解为数据可视化课程实验层面的分析结果，而不是正式天体物理科学结论。
